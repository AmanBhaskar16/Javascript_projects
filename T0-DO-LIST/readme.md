# Todo App

A simple todo list web app with add, edit, and delete functionality. Uses localStorage for persistence.

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Saves todos in localStorage

## Technologies
- HTML
- CSS
- JavaScript (Vanilla)

## How to Run
1. Open `index.html` in your browser.
2. Use the input box to manage your todos.
